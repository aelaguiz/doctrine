Default producer inheriting both shared rules and the default turn sequence.

## Shared Rules

### Ledger Protocol

Keep the ledger current at the start and end of every turn.
Name the current files so the next owner can read them first.

### Turn Result Contract

Every turn should end with one clear handoff and stop.
Do not hand off weak or half-done work.

## How To Take A Turn

### Turn Sequence

Read the active issue and the current files.
Do only this lane's job.
Leave one clear handoff and stop.
