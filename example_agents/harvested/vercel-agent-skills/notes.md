# Notes

## Source Summary

This source mixes a repo-level AGENTS guide with compiled skill docs for React,
React Native, view transitions, and composition patterns. It is useful for
understanding how long-form markdown instruction sets are packaged, generated,
and consumed as agent-facing behavioral contracts.

## High-Signal Patterns

- Repo instructions cover skill creation, build, validate, and extraction
  commands directly.
- Compiled AGENTS docs present rule catalogs as the generated source of truth.
- Skill docs distinguish when to apply the skill, what to optimize, and how to
  avoid regressions.
- View transitions encode trigger order, motion semantics, and browser support
  as an explicit workflow.
- Composition patterns show how to split state, variants, and compound
  components into reusable markdown guidance.
- React Native guidance adds mobile-specific rendering, list, animation, and
  monorepo constraints.

## What This Teaches Doctrine

This source is useful for Doctrine examples that need:

- generated markdown instruction trees with clear source and output layers
- rule catalogs that map to categories and prefixes
- performance-focused guidance with explicit bad and good patterns
- domain-specific constraints for React, React Native, and view transitions
- markdown-native skill packaging conventions
